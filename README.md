# MediDB — Clinical Decision Support System

> Hệ thống hỗ trợ quyết định lâm sàng — tra cứu tương tác thuốc dựa trên DrugBank v5

## 🌐 [▶ Xem giao diện trực tiếp](https://wewefeef.github.io/DrugHome/)

---

### Tính năng
- Tra cứu **17,430 thuốc** từ DrugBank v5
- Xem chi tiết **5,206 protein/target**
- Kiểm tra tương tác thuốc với mạng lưới SVG 3D
- Tính điểm rủi ro 0–10 dựa trên CYP enzyme

### Stack
| Layer | Tech |
|---|---|
| Frontend | React 19 · Vite · TypeScript · Tailwind CSS |
| Backend | FastAPI · SQLAlchemy · MySQL |
| Data | DrugBank® v5 — licensed for academic use |
